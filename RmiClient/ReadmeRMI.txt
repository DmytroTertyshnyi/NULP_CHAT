1. Download and latest version of Maven.zip
2. Unzip Maven.
2. Set the environment variable JAVA_HOME to C:\Program Files\Java\jdk....
3. Set the environment variable MVN_HOME to Maven path(optional).
4. Add to "Path" : %JAVA_HOME%/bin; and  %MVN_HOME%/bin;

/// Steps(1-4) are need to run the ".bat" file "start.bat" to start RMI server. ///

5. Pull all data from “Seminar3” branch to your PC.
6. Import project into Eclipse workspace.
7. Run java class with main method.
